Notes related to Tailwind CSS

Commands to generate proper production development environment to use Tailwind CSS (Setting up Tailwind CSS)

  1. Create a project folder where you want to store your project.
  2. Create an index file with .html like index.html
  3. Create a style file with .css like style.css
  4. Inside the style.css file mention these codes (These are the custom post css directive provided by the Tailwind): 
                        
@tailwind base;
@tailwind components;
@tailwind utilities;

  5. Before installing npm packages you need to install Node.js on your system. then start installing few packages from npm for Tailwind CSS

    Create package JSON file

                    
npm init -y

    Get some more packages (we will put all in our dev dependencies)

    Front-end build tool vite. It'll work nicely here, because vite is going to take existing index.html file as an entry point. It's going to run Tailwind through PostCSS for us and also give us a dev server that updates every time we make changes. 

                    
npm install -D tailwindcss postcss autoprefixer vite

  6. Now its time to edit package.json file's script there you will get "test": "echo \"Error: no test specified\" && exit 1" now just replace "test" with "dev" and "echo \"Error: no test specified\" && exit 1" with "vite".

    Now configure Tailwind with PostCSS

    Generate two file with npx (-p for postcss config)

                        
npx tailwindcss init -p
  7. Now link your style.css or fileName.css file with your index.html or fileName.html (Snippet generator tool) 
                            
<link rel="stylesheet" href="./css/styles.css">
  8. At last you need to edit tailwind.config.js there inside content: [""] mention content: ["*"]

    What we did here

  • Generated own version of Tailwind with a css entry file
  • Then we setup our front-end tooling using vite
  • We installed three other dependencies
    • tailwindcss
    • postcss
    • autoprefixer
  • We created the dev script that runs the vite server
  • Then we generated default Tailwind config file then PostCSS config file
  • Then be able to work in our html file we added link of css file to our html file.
